# Adam's Dotfiles
My dotfiles with some scripts to make installation and updating easy.

Currently running NeoVim + Tmux using iTerm2 nightly in OSX Sierra.


## Setup
- `$ git clone https://github.com/adambiggs/dotfiles.git ~/.dotfiles`

## Install
- `$ cd ~/.dotfiles`
- `$ cli/install`

## Update
- `$ cd ~/.dotfiles`
- `$ cli/update`
